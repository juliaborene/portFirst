Sobre mim

Em 2022, decidi aprimorar minhas habilidades em design a uma profissão, até que cai de cabeça na codificação e desenvolvimento Front-end. Avançando até hoje, tive o privilégio de criar sites e designs para pessoas com grande carisma, onde vi que meu trabalho é valorizado.

Meu foco principal atualmente é construir interfaces de usuário acessíveis para os clientes. Eu gosto mais de criar estruturas no ponto ideal onde o design e a engenharia se encontram - coisas que parecem boas, mas também são construídas bem nos bastidores. No meu tempo livre, investigo como é feito alguns designs e interfarces, para aprimorar minhas habilidades.

No momento, estou expandindo meus conhecimentos para incluir a linguagem React, reconhecendo sua importância no cenário atual de desenvolvimento web e explorando as possibilidades que o React oferece para a criação de interfaces de usuário dinâmicas e eficientes.

Quando não estou no computador, geralmente estou trilhando, jogando jogos online ou até conversando com amigos.
